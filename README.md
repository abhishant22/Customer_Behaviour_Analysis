📊 Data Analytics Project – Customer Behavior Analysis

🔍 Overview

This project focuses on analyzing customer shopping behavior using a dataset of transactions. 
The goal is to extract meaningful insights such as spending patterns, customer segmentation, product preferences, and subscription trends to support better business decisions.

📁 Dataset
Contains ~3,900 customer transactions
Includes details like:
Customer ID
Product Category
Purchase Amount
Subscription Status
Purchase Date
Data is stored in CSV format and imported into Python and SQL databases.
🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
SQL (PostgreSQL / MySQL / SQL Server)
Power BI (Dashboard creation)
Jupyter Notebook (Analysis & EDA)
Gamma (Presentation/PPT creation)

⚙️ Project Steps
1. Data Loading
Loaded dataset using Pandas
Connected SQL database using Python (psycopg2 / SQLAlchemy)
2. Data Cleaning
Removed missing/null values
Handled duplicates
Converted data types (dates, numeric fields)
Standardized column names
3. Exploratory Data Analysis (EDA)
Analyzed customer spending patterns
Identified top product categories
Visualized trends using graphs and charts
Studied subscription behavior
4. SQL Analysis
Imported dataset into SQL database
Wrote queries to:
Aggregate sales
Filter customer segments
Identify top-performing categories
Analyze monthly trends
5. Dashboard Creation (Power BI)
Built interactive dashboard including:
Total revenue
Category-wise sales
Customer segmentation
Subscription insights
6. Reporting
Summarized key findings
Highlighted business insights
Created a clean, structured report
7. Presentation
Designed a professional PPT using Gamma
Included visuals, insights, and conclusions
